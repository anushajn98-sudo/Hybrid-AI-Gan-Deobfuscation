About Our Technology

Our system uses multiple advanced approaches to detect and analyze potentially malicious code:

Triple Detection Models:

  • LSTM Neural Network: Specialized in pattern recognition for sequential code data
  • CodeBERT Transformer: Advanced transformer model fine-tuned for code analysis
  • GAN Classifier: Generative Adversarial Network for sophisticated malware detection

Comprehensive Analysis:

  • Google Gemini AI: State-of-the-art AI for code deobfuscation and execution analysis
  • AST Analysis: Abstract Syntax Tree parsing for structural understanding
  • Lexical Analysis: Token-level examination of code components
  • BLEU Score: Quantitative similarity measurement between different deobfuscations
  • Code Execution: Safe execution of deobfuscated code to verify functionality

All three models (LSTM, CodeBERT, and GAN) work together with Gemini AI to provide comprehensive malware detection and analysis.
